# FFmpeg for Unity

## Overview

This assets is to execute FFmpeg(http://ffmpeg.org/) on editor and application.

You can do below:

- Play Movies(mp4, avi, mov etc.)
- Capture Game View or Camera(In Game)
- Convert Video Files
- Live Streaming to Youtube(https://www.youtube.com/)(rtmp etc.)
etc.

## Correspondence Environment

- Unity Editor(Windows/Mac/Linux)
- Standalone(Windows/Mac/Linux)(Mono/IL2CPP)
- Android
- iOS

## Note for FFmpeg

This assets uses FFmpeg binary(Windows/Mac/Linux) or plugins(Android/iOS) built below.
For windows, Please import or build below:

https://github.com/NON906/ffmpeg-windows-build-helpers/releases/

For others, using libraries is part only(please show Lisense). So, if libraries you want to use is not included, please rebuild and replace files. 

- Mac/Linux(https://github.com/NON906/ffmpeg-build-script)
- Android/iOS(https://github.com/NON906/ffmpeg-kit)
